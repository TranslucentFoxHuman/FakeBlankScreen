# FakeBlankScreen
このアプリは、起動すると真っ黒い画面を表示して、画面のWakeLockを有効にしスリープを防ぎます。OLEDスクリーンを搭載したデバイスを対象としています。

あなたはOLEDスクリーンを搭載したデバイスを所有していて、その上で何かバックグラウンドで時間のかかる処理をしていますか? そしてその処理を行ないつつもバッテリー消費を抑え、画面の焼き付きを防ぎたいですか? このアプリはそういったケースに最適です!

黒い画面を表示しておくことで、デバイスをスリープ状態にするのと異なりバックグランドでの処理が停止/制限される事はありません。  
さらに、ただ画面をオンにしておくのと異なり、OLEDスクリーンが焼き付く事もありません。

## 使用方法
アプリを起動すると、真っ黒い画面が表示されます。そして画面のWakeLockを有効にしデバイスがスリープモードに入るのを防ぎます。それだけです。

アプリを閉じるには、画面の上下の端からスワイプしてナビゲーションバーを表示し、四角いボタンからアプリ履歴を表示してこのアプリを閉じてください。アプリ自体にはアプリを終了する機能はありません。

## Tips
### 私はこのアプリを使用中にスマートフォンのそばから離れる必要がありますが、安全にしておきたい。どうすれば良い?
[Androidのアプリ固定機能](https://support.google.com/android/answer/9455138)を使用すると良いでしょう。アプリをピン留めすると、通知欄を開いたり、アプリを終了したり、アプリを切り替えたりといった操作を防ぎます。アプリを切り替えようとすると認証が必要になるため、安心してスマートフォンのそばを離れることができます。

### LCDスクリーンでも使えますか?
動作はしますが、まったく役には立ちません。OLEDスクリーンでは、黒を表示している間は画面の消費電力はゼロになりますが、LCDスクリーンではバックライトが点灯し続けるため消費電力は高いままです。せいぜい、LCDのドット抜けやバックライトの斑を見付けるのには使えるかもしれません。

### なぜバージョン2.x.x? 1.x.xではないのですか?
1.xは厳密には存在しています。(gitのコミット履歴を見てください:e71a3b3dab8f22fdab1a17f7b2e09c88302e56c8)
しかしこれは当時、私自身のためだけに開発されたもので、Android 13以降でなければ動作しませんでした。  
またこれは2022年に開発されたもので非常に古い。アプリそのものが単純なので、既存のコードを読み直して改善するよりは、ゼロから新しいプロジェクトとして書き直した方が良いと考えました。  
そのため既存のプロジェクトファイルを削除し、全く新しいアプリとして書き直しました。これによってコードの互換性が無くなったため、2.x.xとしてバージョンが付けられました。
